# Fractol
Fractal graphic 42 project

> Fractal Generator program

### How to use

`make`

`Usage : ./fractol fractal_name [-w window width] [-h window height]`

> print help
`./fractol --help`

### Fractal names
* julia
* mandelbrot
* burningship
